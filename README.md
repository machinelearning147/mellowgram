# Mellowgram Prototype (Vite + React + Tailwind) - Upload-enabled

## Quick start

1. Extract this folder.
2. Run:
   npm install
   npm run dev
3. Open http://localhost:5173 in your browser and resize to mobile width (or use device emulation).

This prototype stores uploaded images locally in the browser (IndexedDB via localforage). For production sync, integrate with a backend and object storage (S3/R2).

Dev dependencies: Node 18+, npm.
